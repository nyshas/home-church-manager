    Tasks
        Creating a Home Group
            1. Open task bar hamburger menu in top left screen
            2. Select Home Groups
            3. Select green "Add Home Group" button
            4. Fill out fields and press "Find Location"
            5. Press "Save Home Group"
        Editing a Home Group
            1. Open task bar hamburger menu in top left screen
            2. Select Home Groups
            3. Press light blue edit button in desired Home Group's row
            4. Edit desired fields
            5. Press Save Home Group
        Add Members
            1. Open task bar hamburger menu in top left screen
            2. Select Members
            3. Select green "Add Member" button
            4. Fill out fields and press "Save Member"
            5. Press "Save Home Group"
